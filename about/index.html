Subscribe Openly

About

Linking directly to a web feed like an RSS file on your website is not ideal for users.

Browsers like Firefox and Chrome will show the raw XML of your feed.

What if there was a better experience?

With Subscribe Openly, you can link to a URL that will show buttons for all the common feed readers.

When clicked, the button will take the user into a view that lets them subscribe to your feed in their preferred reader.

How To

To use Subscribe Openly, add the following to any links to your web feed on your site:

https://subscribeopenly.net/subscribe/?url=YOUR_FEED_URL
